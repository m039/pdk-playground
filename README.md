# Varios Examples With Padauk MCU

In this repository, you can find varios examples which are using [pdk-platformio](https://github.com/m039/pdk-platformio).

1. Blink - basic example of blinking a led
2. BlinkAsm - the same, but in assembler
3. Led PWM - basic example of using PWM for fading a led
4. Musical Melody (Period Mode) - a project contains code that plays mostly Russian melodies on a buzzer
5. Musical Melody (PWM Mode) - a project for testing adjusting volume by changing PWM signal
